Python is an open-source, object-oriented, high-level programming language.  It was developed by **Guido van Rossum** in 1991. It is presently owned by Python Software Foundation (PSF).

### Installing Python
To install Python, we must first download the installation package of the required version from the following link:
[Download Python](https://www.python.org/downloads/)
>**Tip:** Instead of using Python IDLE install editors like [VS Code](https://code.visualstudio.com/download), [Spyder IDE](https://www.spyder-ide.org/) or [PyCharm](https://www.jetbrains.com/pycharm/download/).
